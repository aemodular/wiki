## Overview

[filename](../../common/micro.md ':include')

This module is a clone of the Rim Shot voice from the Boss DR-55 drum machine. The pitch-mod has been incorporated into the design for some flexibility

## Controls

* **pitch** Changes the pitch of the tone, at the higher end it sounds more like a clave.

## Connectivity

As a drum voice, this one is fairly self-explanatory!

### Inputs

* **trig** A pair of trigger inputs (1 & 2) are provided which sum so that accented hits can be achieved. Each trigger is actually a pair of sockets acting as a local multiple.

### Outputs

* **out** the audio from the module.
